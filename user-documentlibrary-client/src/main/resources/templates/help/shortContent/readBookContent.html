In the read book function you can change page in many way. You can change page with the keyboard arrows key, with the command on the tool bar and with the command in the following tool bar. For other details please follow next tips
Change page by arrows Key
Change page by tool bar command
  • for enable/disable bookmark during read you can click on the button like this on the tool bar
  • for change page during read you can click on the buttons like those on the tool bar
  • For change page in a direct way you can use the command on the tool bar
Change page by following tool bar command
  • for enable/disable bookmark during read you can click on the button like this on the tool bar
  • for change page during read you can click on the buttons like those on the tool bar
  • For change page in a direct way you can use the command on the tool bar
