If you want upload a new book you can choice the upload book function on the menu as you can see in the figure. For other details please follow next tips

For upload a new book you must choose a pdf as book master and optionally Author, Description and Search Tags. For information about the field you can use the tool tips ad in figure

Once you have click on choose file the system will open a file chooser. In this stage you have choose the pdf file that you want upload

For submit the form data, file and metadata you must click on the button in highlighted in figure

After that you submitted the form you have wait for the task completion. For know if the task is finished, you can wait the notification as highlighted in figure. Clicking on the notification icon the system will open a container with all the notification and you can choice of delete the notification, publish or delete on the fly the book
